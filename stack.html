Stack Data structure

A stack is a linear data structure that follows the principle of Last In First Out (LIFO).This means last element inserted the stack is removed first

LIFO Principle of stack

In programming term, putting an item on top of the stack is called Push and removing an item is called POP

Basic Operations of Stack

There are some basic operations that allow us to perform different action on a stack

  • Push : Add an element to the top of a stack
  • Pop : Remove an element from the top of stack
  • IsEmpty : Check if the stack is empty
  • IsFull: Check if the stack full
  • Peek : Get the value of the top element without removing it
