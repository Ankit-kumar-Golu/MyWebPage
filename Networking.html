Networking

A Computer is a set of computers sharing resources located on or provided by network nodes.

Types of Networks

  1. Division based on the communication
    • Wired Network
    • Wireless Network
  2. Division based on area covered
    • Local Area Network (LAN)
    • Metropolitan Aera Network (MAN)
    • Wide Area Network (WAN)
  3. Based on types of communication
    • Ponit TO Ponit networks
    • Broadcast networks
  4. Based on types of architecture
    • P2P Networks
    • Client - Server Networks
    • Hybrid Network
